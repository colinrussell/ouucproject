CanadaEh is a showcase of Canadian information and entertainment. Find useful conversion tools as well as comedy here!

In Canada, the metric system is used.

Distance is measured in kilometers, temperature is measured in Celsius,
liters are used for measuring liquids, and mass is measured in kilograms.
